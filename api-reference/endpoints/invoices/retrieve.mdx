---
title: 'Retrieve Invoice'
api: 'GET /api/v1/invoices/{id}'
description: 'Retrieve a specific invoice'
---

## Request

<ParamField path="id" type="string" required>
  The invoice ID (CzUid) to retrieve
</ParamField>

<ParamField query="expand[]" type="array">
  Expand nested objects. See [Expanding Objects](/concepts/expanding-objects) for details.
  
  Available expansions:
  - `lines` - Invoice line items (included by default)
  - `payments` - Payment records (included by default)
  - `client` - Client details
  - `efactura` - E-Factura submission info
</ParamField>

<Note>
  By default, retrieving a single invoice automatically expands the `lines` and `payments` objects for convenience.
</Note>
