---
title: 'List Products'
api: 'GET /products'
description: 'Retrieve a paginated list of products'
---

## Request

<ParamField query="page" type="integer" default="1">
  Page number to retrieve
</ParamField>

<ParamField query="per_page" type="integer" default="50">
  Number of items per page (max: 100)
</ParamField>

<ParamField query="search" type="string">
  Search term to filter by product name, code, or barcode
</ParamField>

<ParamField query="category_id" type="integer">
  Filter by category ID
</ParamField>

<ParamField query="is_active" type="integer">
  Filter by active status
  - `1` - Active products only
  - `0` - Inactive products only
</ParamField>

<ParamField query="sort" type="string" default="name">
  Field to sort by
  - `name` - Product name
  - `price` - Selling price
  - `created_at` - Creation date
</ParamField>

<ParamField query="order" type="string" default="asc">
  Sort order. Options: `asc`, `desc`
</ParamField>

<RequestExample>

```bash cURL
curl https://api.contazen.ro/v1/products \
  -H "Authorization: Bearer sk_live_YOUR_API_KEY"
```

```bash cURL with Filters
curl https://api.contazen.ro/v1/products \
  -H "Authorization: Bearer sk_live_YOUR_API_KEY" \
  -G \
  -d search=laptop \
  -d is_active=1 \
  -d sort=price \
  -d order=desc
```

```javascript Node.js
const response = await fetch('https://api.contazen.ro/v1/products?' + 
  new URLSearchParams({
    search: 'laptop',
    is_active: '1',
    page: '1',
    per_page: '20'
  }), {
  headers: {
    'Authorization': 'Bearer sk_live_YOUR_API_KEY'
  }
});

const data = await response.json();
```

```python Python
import requests

response = requests.get(
    'https://api.contazen.ro/v1/products',
    params={
        'search': 'laptop',
        'is_active': 1,
        'sort': 'price',
        'order': 'desc'
    },
    headers={'Authorization': 'Bearer sk_live_YOUR_API_KEY'}
)

data = response.json()
```

```php PHP
$ch = curl_init();
$params = http_build_query([
    'search' => 'laptop',
    'is_active' => 1,
    'page' => 1
]);

curl_setopt($ch, CURLOPT_URL, 'https://api.contazen.ro/v1/products?' . $params);
curl_setopt($ch, CURLOPT_HTTPHEADER, [
    'Authorization: Bearer sk_live_YOUR_API_KEY'
]);
curl_setopt($ch, CURLOPT_RETURNTRANSFER, true);

$response = curl_exec($ch);
$data = json_decode($response, true);
```
