---
title: 'Retrieve Client'
api: 'GET /clients/{id}'
description: 'Retrieve a specific client by ID'
---

## Request

<ParamField path="id" type="string" required>
  The client ID
</ParamField>

## Response

<ResponseField name="success" type="boolean">
  Indicates if the request was successful
</ResponseField>

<ResponseField name="data" type="object">
  The client object
</ResponseField>

<ResponseField name="meta" type="object">
  Response metadata
</ResponseField>

<RequestExample>

```bash cURL
curl https://api.contazen.ro/v1/clients/1a2b3c4d5e \
  -H "Authorization: Bearer sk_live_YOUR_API_KEY"
```

```javascript Node.js
const response = await fetch('https://api.contazen.ro/v1/clients/1a2b3c4d5e', {
  headers: {
    'Authorization': 'Bearer sk_live_YOUR_API_KEY'
  }
});

const data = await response.json();
```

```python Python
import requests

response = requests.get(
    'https://api.contazen.ro/v1/clients/1a2b3c4d5e',
    headers={'Authorization': 'Bearer sk_live_YOUR_API_KEY'}
)

data = response.json()
```

```php PHP
$ch = curl_init('https://api.contazen.ro/v1/clients/1a2b3c4d5e');
curl_setopt($ch, CURLOPT_HTTPHEADER, [
    'Authorization: Bearer sk_live_YOUR_API_KEY'
]);
curl_setopt($ch, CURLOPT_RETURNTRANSFER, true);

$response = curl_exec($ch);
$data = json_decode($response, true);
```
