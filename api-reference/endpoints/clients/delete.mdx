---
title: 'Delete Client'
api: 'DELETE /api/v1/clients/{id}'
description: 'Delete a client'
---

## Request

<ParamField path="id" type="string" required>
  The client ID to delete
</ParamField>

<Warning>
  **Important**: You can only delete clients that have no associated invoices. If the client has any invoices, the deletion will fail with an error.
</Warning>

## Response

<ResponseField name="success" type="boolean">
  Indicates if the request was successful
</ResponseField>

<ResponseField name="data" type="object">
  <Expandable title="Properties">
    <ResponseField name="deleted" type="boolean">
      Always `true` when deletion is successful
    </ResponseField>
    
    <ResponseField name="id" type="string">
      The ID of the deleted client
    </ResponseField>
  </Expandable>
</ResponseField>

<ResponseField name="meta" type="object">
  Response metadata
</ResponseField>

<RequestExample>

```bash cURL
curl -X DELETE https://app.contazen.ro/api/v1/clients/9z8y7x6w5v \
  -H "Authorization: Bearer sk_live_YOUR_API_KEY"
```

```javascript Node.js
const deleteClient = async (clientId) => {
  const response = await fetch(`https://app.contazen.ro/api/v1/clients/${clientId}`, {
    method: 'DELETE',
    headers: {
      'Authorization': 'Bearer sk_live_YOUR_API_KEY'
    }
  });

  return await response.json();
};
```

```python Python
import requests

def delete_client(client_id):
    url = f'https://app.contazen.ro/api/v1/clients/{client_id}'
    headers = {
        'Authorization': 'Bearer sk_live_YOUR_API_KEY'
    }
    
    response = requests.delete(url, headers=headers)
    return response.json()
```

```php PHP
<?php
$clientId = '9z8y7x6w5v';
$url = 'https://app.contazen.ro/api/v1/clients/' . $clientId;

$ch = curl_init($url);
curl_setopt($ch, CURLOPT_CUSTOMREQUEST, 'DELETE');
curl_setopt($ch, CURLOPT_HTTPHEADER, [
    'Authorization: Bearer sk_live_YOUR_API_KEY'
]);
curl_setopt($ch, CURLOPT_RETURNTRANSFER, true);

$response = curl_exec($ch);
$result = json_decode($response, true);
?>
```
